Proyecto web de casino Flask - 
- Registro y login de usuarios con verificación de correo
- Sesión activa y persistencia mediante `Flask` y `session`
- Sistema de saldo virtual con posibilidad de recarga
- Juego de Blackjack usable  y visualización de cartas
- Juego de ruleta en proceso
- Uso de archivos JSON para almacenar usuarios e historial



codigo de ejecucion en la terminal 
python app.py
